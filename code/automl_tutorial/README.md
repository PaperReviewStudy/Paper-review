## Install NNI

```bash
pip install git+https://github.com/microsoft/nni
```

### Run AutoML Experiment

```bash
nnictl create -c nni.yml
```

## Install sacred

```bash
pip install sacred
```

## Install omniboard
```bash
npm install -g omniboard
omniboard -m hostname:port:database

```
### Local
```bash
omniboard -m 127.0.0.1:27017:experiments
```
